cd <pasta>  -> entra na pasta
mkdir <pasta> -> cria uma nova pasta com o nome <pasta>
ls [-la] (linux) -> lista os arquivos dentro da <pasta> qaue você está
dir [/a] (windows) -> lista os arquivos dentro da <pasta> qaue você está
pwd -> mostra a <pasta> que você está agora
clear/cls (ou ctrl+L) -> limpa a tela


# Comandos GIT

(git) init -> inicia um repositório git na <pasta> que você está agora.
